Archivo plano

Un archivo de texto simple, texto sencillo o texto sin formato, es un archivo informático que contiene únicamente texto formado solo por caracteres que son legibles por humanos y carece de cualquier tipo de formato tipográfico

Archivo XML.

El Extensible Markup Language (XML) o Lenguaje Extensible de Etiquetado es definido como un lenguaje marcado para documentos que contienen información estructurada que posee contenido (como palabras o imágenes) y alguna indicación del papel que desempeña.

Archivo tipo JSON

Los archivos tipo json (JavaScript Object Notation), es un formato el cual es implementado por Java. Hace parte de uno de los lenguajes más importantes de la programación y tiene como fin la creación de páginas web como también en la generación de múltiples software.

Archivo Parquet



Apache Parquet es un formato de archivo en columnas que proporciona optimizaciones para acelerar las consultas. Es un formato de archivo mucho más eficaz que el archivo .csv o JSON.
